Скрипты, представленные в репозитории, демонстрируют парсинг сайтов с прокручиванием страницы:

1)Скрипт прокручивает учебную страницу до получения нужного элемента с паролем и выводит результат в консоль.

2)Парсер заполняет динамически появляющиеся поля учебного сайта и возвращает в консоли пароль, который появляется в конце страницы также динамически.

3)Скрипт демонстрирует работу с перетаскиванием элемента в заданную область.

4)Скрипт выполняет автоматическое двойное нажатие левой кнопкой мыши на элемент.

5)Скрипт передает сайту информацию о нажатии клавиш.

6)Скрипт демонстрирует работу с контекстным меню.

7)Скрипт демонстрирует прокрутку элементов страницы.

8)Скрипт демонстрирует прокрутку страницы до нужного элемента.

9)Скрипт демонстрирует прокрутку страницы и сбор информации из одинаковых элементов.

10)Скрипт демонстрирует прокрутку элемента страницы и сбор информации из одинаковых элементов.

11)Скрипт демонстрирует прокрутку элемента страницы и сбор информации из одинаковых элементов.

12)Скрипт демонстрирует прокрутку нескольких контейнеров страницы и сбор информации из одинаковых элементов.

13)Скрипт демонстрирует сбор элементов из разных частей веб-страницы.

14)Скрипт демонстрирует поочередное нажатие и удерживание кнопок веб-страницы.

15)Скрипт отмечает в контейнере только чекбоксы с четным значением атрибута value.

16)Скрипт автоматически вставляет лайки в комментарии сайта и собирает появившиеся числа, на выходе выдает сумму чисел.
